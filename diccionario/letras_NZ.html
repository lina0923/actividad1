Palabras N - Z

N

Nodo: Elemento de una estructura de datos, como una lista o un árbol.

Namespace: Espacio de nombres que organiza y evita conflictos entre identificadores.

Normalización: Proceso para optimizar bases de datos y evitar redundancias.

O

Operador: Símbolo utilizado para realizar operaciones matemáticas o lógicas.

Open Source: Software con código abierto que puede modificarse y distribuirse libremente.

Overflow: Error que ocurre cuando un valor excede la capacidad de almacenamiento.

P

Programación: Proceso de crear software mediante lenguajes de programación.

Parámetro: Variable utilizada para pasar información a funciones y métodos.

Proceso: Ejecución de un programa en un sistema operativo.

Q

Query: Consulta utilizada en bases de datos para recuperar información.

QoS: Calidad del servicio en redes que controla el tráfico de datos.

Queue: Estructura de datos tipo cola, donde el primer elemento en entrar es el primero en salir.

R

Repositorio: Almacén donde se guarda y gestiona código fuente.

Refactorización: Mejora del código sin cambiar su funcionalidad.

RAM: Memoria de acceso rápido utilizada para ejecutar programas.

S

Software: Conjunto de programas y sistemas que hacen funcionar un dispositivo.

Servidor: Computadora o sistema que proporciona servicios a otros dispositivos.

Seguridad: Prácticas y técnicas para proteger información y sistemas de amenazas.

T

Thread: Hilo de ejecución dentro de un proceso que permite realizar múltiples tareas simultáneamente.

Tipo de Datos: Clasificación de valores en programación, como enteros, cadenas y booleanos.

Token: Identificador en programación y seguridad usado para autenticar usuarios.

U

Usuario: Persona o entidad que interactúa con un sistema informático.

URL: Dirección única que identifica recursos en la web.

Unidad: Elemento individual dentro de un sistema de almacenamiento.

V

Variable: Espacio en memoria que almacena valores que pueden cambiar.

Virtualización: Tecnología que permite ejecutar múltiples sistemas operativos en un solo hardware.

Vector: Estructura de datos utilizada para almacenar múltiples valores en una dimensión.

W

Web: Conjunto de páginas y servicios accesibles a través de internet.

Wi-Fi: Tecnología inalámbrica para conectar dispositivos a internet.

Wrapper: Código que encapsula otro código para modificar o extender su funcionalidad.

X

XML: Lenguaje de marcado utilizado para estructurar y almacenar datos.

XPath: Lenguaje para navegar y consultar datos dentro de documentos XML.

XOR: Operador lógico que devuelve verdadero si los valores son diferentes.

Y

YAML: Formato de serialización de datos legible para humanos.

Yield: Palabra clave en programación utilizada para generar valores en una función.

Year 2000 Bug: Error causado por el almacenamiento de años con dos dígitos en sistemas antiguos.

Z

ZIP: Formato de compresión de archivos utilizado para reducir su tamaño.

Zero-day: Vulnerabilidad desconocida explotada por hackers antes de que se publique un parche.

Zona de memoria: Espacio en la RAM asignado a procesos en ejecución.
